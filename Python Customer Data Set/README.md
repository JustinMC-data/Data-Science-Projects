# Customer Purchase Behavior Visualization Project

## Overview

This project explores customer purchasing behavior using Python-based
data visualization. The goal is to identify high-value customers and
compare spending behavior against key attributes such as income,
membership duration, and purchase frequency.

The analysis generates visual insights that can support: - Customer
segmentation strategies - Loyalty program targeting - Marketing
optimization - Revenue opportunity identification

This project demonstrates applied skills in: - Data manipulation -
Exploratory data analysis (EDA) - Visualization design - Script-driven
analytics workflows

## Visual Analyses Included

### 1. Top Customers by Spending Score vs Income

Compares the highest-spending customers against the highest-income
customers to explore whether spending intensity correlates with income.

### 2. Spending Score vs Membership Duration

Evaluates whether longer customer relationships correlate with higher
spending behavior.

### 3. Spending Score vs Purchase Frequency

Compares high spenders with frequent purchasers to assess behavioral
overlap.

### 4. Top 10 Customers by Spending Score

Standalone visualization highlighting highest-value customers.

## How to Run

### Install Dependencies

    pip install pandas matplotlib seaborn

### Execute Script

    python Combined_visuals.py

### Menu Options

    1 — Spending vs Income
    2 — Spending vs Membership
    3 — Spending vs Frequency
    4 — Top Spenders
    5 — Run All
    6 — Exit
